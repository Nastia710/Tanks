# Файли із завдання "Корисна інформація"
Всі скрипти, які розміщенні у завданні "Корисна інформація", розміщені у папці Useful links.

# Модифікації
Із особистих модифікацій додано:
1. SkyBox змінено на зіркове небо (асети зіркового неба виявилися по 192 МБ кожний, тому проект на гіт був залитий за допомогою Git LFS).
2. Коли здоров'я танка стає меншим за 50% підключаються частинки іскор та диму для візуального відображення пошкодженості танка (інтенсивність іскор залежить від значення пошкодження ratio).
3. Додано бочки, які взриваються при попаданні в них снарядом. Знімають здоров'я танка в залежності в якому радіусі від бочки знаходився танк (чим ближче, тим більший урон).
4. Кожен танк може виставляти міни на полі. Якщо у меню, при виборі танку, вибрано P1 (тобто керування танком здійснюється AWSD), то міна ставиться клавішею 1; якщо ж у меню, при виборі танку, вибрано P2 (тобто керування танком здійснюється стрілками), то міна ставиться клавішею RightSpace. Танки, які керуються ШІ, кидають міну в рандомний момент часу. Міна має 1,5 секунди перед тим як стати активною для вибуху, щоб танк, який її "поклав" не підірвався на ній. Бомба активується лише коли на нею наїде танк.
5. Додано місця для відновлення здоров'я на +50 очок. Танк заїжджає в це місце (не обов'язково повністю, головне аби торкався його) і танк повинен торкатися цього спрайта прягом 4,5 секунд, щоб отримати до свого здоров'я +50 очок здоров'я. При цьому танк може рухатися, але обов'язково повинен торкатися місця відновлення здоров'я. Коло цього місця, поки танк його торкається ці 4,5 секунди, поступово заповнюється кольором. Якщо час "надавання" здоров'я не закінчився, а танк виїхав із зони, то шкала (колір) і час відліку тих 4,5 секунд зони здоров'я обнуляється і ніякі очки до здоров'я танку не додаються. Якщо танк всі ці 4,5 секунди торкався спрайта (і в цей час як раз колір заповнився повністю), то відразу танку даються ці 50 очок до здоров'я (його шкала здоров'я з кольором також заповнюється кольором до правильного моменту), а час і колір спрайту, який надава здоров'я обнуляються. Якщо здоров'я надано (танк вистояв повністю всі ці 4,5 секунди), то зона здоров'я зникає назавжди.
